# AI Validation Checklist

This checklist must be executed before accepting code generated by AI.

---

# Domain Integrity

Check:

[ ] Did the AI modify domain entities?

[ ] Did the AI modify entity attributes?

[ ] Did the AI introduce new entities?

---

# State Machine Integrity

Check:

[ ] Did the AI add new states?

[ ] Did the AI change transition rules?

[ ] Did the AI bypass state validation?

---

# Database Integrity

Check:

[ ] Did the AI modify the schema?

[ ] Did the AI add new tables?

[ ] Did the AI change ENUM values?

---

# Financial Logic

Check:

[ ] Did the AI change payment logic?

[ ] Did the AI modify refund rules?

[ ] Did the AI introduce new payment types?

---

# Event Integrity

Check:

[ ] Did the AI invent new events?

[ ] Did the AI modify event payloads?

[ ] Did the AI bypass event triggers?

---

# Final Decision

If **any answer is YES**, the generated code must be rejected and reviewed manually.
